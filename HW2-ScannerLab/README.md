# Lexical Analyzer in C

Reads text files with `"input"` in their name, scans for numbers and basic operators, and writes tokens to a new file (replacing `"input"` with `"output"`).

---

## Requirements
- **C11 or later**
- A C compiler like `gcc`

---

## Compile
```bash
gcc HW2.c -o app
```
---

## Run
```bash
./app
